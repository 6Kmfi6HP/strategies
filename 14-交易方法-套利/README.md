# 14-交易方法-套利

> 套利策略通过捕捉不同市场、不同合约、不同交易所之间的价格差异获利。包括跨市场套利、统计套利、期现套利、三角套利等。理论上为无风险或低风险策略。

---

## 📋 分类特点

- **优势**：理论上风险低、收益稳定
- **适用市场**：多市场、多品种
- **风险等级**：⭐⭐（中低风险）
- **核心要求**：快速执行、资金效率、价差监控
