# 07-技术指标-CCI

> CCI（Commodity Channel Index，顺势指标）是一种震荡指标，用于识别价格偏离统计平均值的程度。CCI 无上下界限，通常以 +100 为超买线、-100 为超卖线。

---

## 📋 分类特点

- **优势**：对价格极端值敏感，适合短线交易
- **适用市场**：震荡市场、短期反转
- **风险等级**：⭐⭐⭐（中等风险）
- **核心信号**：CCI > +100 超买、CCI < -100 超卖
