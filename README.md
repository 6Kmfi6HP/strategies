## 📖 项目介绍

本项目是一个全面的量化交易策略文档库，汇集了来自 FMZ 量化平台等多个来源的 **5,807 个**策略文档。内容涵盖主流技术指标（移动平均线、MACD、RSI、布林带等）、多种交易方法（趋势跟踪、反转交易、网格套利等）、不同资产类型（加密货币、贵金属等）的量化策略。

**技术栈**：PineScript、JavaScript、Python、Pine Script v5
**适用对象**：量化交易者、策略开发者、金融研究人员、算法交易学习者

**核心特点**：
- ✅ 策略文档结构完整：包含策略原理、源代码、优化方向、风险分析
- ✅ 多语言支持：中英文双语说明
- ✅ 实战导向：每个策略均包含可运行的代码实现
- ✅ 系统分类：26 个专业分类，便于快速检索

## 🚀 使用说明

### 1. 如何浏览策略

**方法一：按分类浏览**
点击上方表格中的 📂 图标，进入对应分类的详细索引页面，查看该类别下的所有策略文件。

**方法二：按需搜索**
使用 GitHub 搜索功能（快捷键 `/`），输入关键词（如"双均线"、"突破"、"RSI"）快速定位策略。

**方法三：按指标筛选**
- 趋势类指标：查看分类 01-04、10
- 震荡类指标：查看分类 03、07、08
- 成交量指标：查看分类 09

### 2. 文件命名规则

所有策略文件遵循统一命名格式：

```
序号-中文名称-英文名称.md
```

**示例**：
- `001-双均线交叉策略-ema-crossover-strategy.md`
- `052-布林带突破-bollinger-breakout.md`

**规则说明**：
- **序号**：三位数字（001-999），同一目录内独立编号
- **中文名称**：策略的中文描述，使用连字符 `-` 分隔
- **英文名称**：策略的英文描述，全小写，使用 kebab-case 风格
- **扩展名**：统一为 `.md` (Markdown 格式)

### 3. 策略文档结构

每个策略文档通常包含以下部分：

```markdown
# 策略名称

## 策略概述
简要描述策略的核心思想和目标

## 策略原理
详细说明策略的逻辑、指标计算方法、交易信号生成规则

## 策略优势
列举策略的优点和适用场景

## 策略风险
分析潜在风险和可能的失效情况

## 优化方向
提供策略改进建议和参数优化思路

## 策略源码
提供 PineScript/JavaScript/Python 等语言的完整实现代码
```

## 🤝 贡献指南

欢迎贡献新的策略文档！请遵循以下规范：

### 贡献流程

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/new-strategy`)
3. 按照文件命名规则添加策略文档
4. 提交更改 (`git commit -m 'docs: 添加XXX策略'`)
5. 推送到分支 (`git push origin feature/new-strategy`)
6. 创建 Pull Request

### 文档规范

- **命名**：严格遵循 `序号-中文名-英文名.md` 格式
- **分类**：选择最合适的目录，如不确定可放入 `26-其他策略`
- **内容**：必须包含策略概述、原理、源码、风险分析
- **代码**：代码需可运行，并注明运行环境和依赖
- **语言**：建议中英文双语，至少包含中文说明

### 代码规范

- PineScript：遵循 Pine Script v5 规范
- JavaScript：使用 ES6+ 语法
- Python：遵循 PEP 8 规范
