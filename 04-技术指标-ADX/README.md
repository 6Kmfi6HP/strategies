# 04-技术指标-ADX

> ADX（Average Directional Index，平均趋向指标）用于衡量趋势的强度，而非趋势方向。ADX 值越高（通常 >25），表示趋势越强；ADX 值低（<20）表示市场处于震荡或盘整状态。

---

## 📋 分类特点

- **优势**：客观评估趋势强度，避免在震荡市场频繁交易
- **适用市场**：任何市场（用于过滤信号）
- **风险等级**：⭐⭐（中低风险）
- **核心信号**：ADX>25 趋势强劲、ADX<20 震荡市场
