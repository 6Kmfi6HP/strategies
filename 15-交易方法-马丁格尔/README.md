# 15-交易方法-马丁格尔

> 马丁格尔（Martingale）策略源自赌博策略，核心思想是亏损后加倍下注，直到盈利覆盖之前损失。量化交易中表现为：浮亏后加仓摊低成本。高风险高收益策略，需严格控制仓位和止损。

---

## 📋 分类特点

- **优势**：理论上必然盈利（无限资金前提下）
- **适用市场**：震荡市场、小幅回调
- **风险等级**：⭐⭐⭐⭐⭐（极高风险）
- **风险提示**：可能导致爆仓，需严格风控
